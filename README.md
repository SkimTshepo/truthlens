# TruthLens 🔍

TruthLens is a fact-checking web application that allows users to input claims and receive verification results with sources.

## 🚀 Features
- Input a claim
- Returns a verdict (True / False / Unverified)
- Displays source of information

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- API Communication: Fetch API

## ⚙️ How to Run
1. Navigate to backend folder
2. Install dependencies:
   pip install flask flask-cors
3. Run server:
   python app.py
4. Open frontend/index.html in browser
